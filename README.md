# Rhetorik & Logik – Memory + Tabu (Vollversion)

Digitale Vollversion mit Spielfeld, virtuellen Karten, 2–4 Teams, Soundeffekten und lokalem Mehrspieler‑Host.

## Start

```bash
npm install
npm start
```

Öffne dann im Browser:

```
http://localhost:3000
```

## Lokaler Mehrspieler

1. Ein Gerät klickt **Als Host starten**. Der Raum‑Code wird angezeigt.
2. Andere Geräte geben den Code ein und klicken **Raum beitreten**.
3. Der Host synchronisiert das Spiel in Echtzeit.

## Features

- Memory‑Board mit Begriff/Definition‑Paaren
- Tabu‑Challenge mit Steal‑Mechanik
- 2–4 Teams
- Soundeffekte (abschaltbar)
- Erklär‑Timer
- Endscreen & Spielanleitung

## Hinweise

- Der Host bestimmt den Spielstand; Gäste sehen den synchronisierten Stand.
- Bei Verbindungsabbruch bleibt das lokale Spiel weiter spielbar.
